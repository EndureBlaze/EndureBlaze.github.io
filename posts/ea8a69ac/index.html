不是这种方式不好，这种确实从根本解决了问题，但是我是实在懒，所以想了一些外门邪道。然后百度到了可以使用 chcp 65001 命令把控制台的编码切换到 UTF-8，当然你可以参考这篇 百度知道 或者这篇 博文 把编码永久更改，但是我怕会出奇怪的问题，所以在 VS Code 的配置文件找了找，没想到还真有这种骚操作。
